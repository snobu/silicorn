# silicorn
Draw histograms on Pimoroni's [Unicorn HAT] with interesting approach to
temporal resolution


### Pics or it didn't happen
[![Reading Low CPU Usage over SNMP][lowcpu]]

[![Reading High CPU Usage over SNMP][highcpu]]


### Build
    sudo ./make

### Run
    sudo seq 100 | ./silicorn
